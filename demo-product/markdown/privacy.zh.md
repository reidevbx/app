# 隱私權政策

**生效日期：2026年1月1日**

我們非常重視您的隱私。本政策說明我們如何收集、使用和保護您的個人資訊。

## 1. 我們收集的資訊
我們可能會收集以下類型的資訊：
- **帳戶資訊**：如您的姓名、電子郵件地址。
- **使用數據**：您如何與我們的服務互動的資訊。

## 2. 資訊的使用
我們使用收集的資訊來：
- 提供和維護我們的服務。
- 通知您有關服務的變更。
- 提供客戶支援。

## 3. 資訊的分享
除非法律要求或為了提供服務，否則我們不會將您的個人資訊出售或分享給第三方。

## 4. 資料安全
我們採取合理的措施來保護您的資料，但請注意，沒有任何網路傳輸方法是 100% 安全的。

## 5. 聯絡我們
如果您對本隱私政策有任何疑問，請聯繫 rei.pxperf@gmail.com。
